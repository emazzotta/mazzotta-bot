# Mazzotta Bot

My telegram bot.

## Commands

|Command|Action|
|---|---|
|/help|Show this text|
|/say|Ask the bot to say something in a bot voice|
|/sing|Ask the bot to sing something in a bot voice|
|/zhaw|Show current ZHAW stats|

## Getting Started

### Bootstrap

```
# One-off
make bootstrap
```

### Configure

Configure your `.env`-file.

### Start

```
make start
```
