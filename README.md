# Mazzotta Bot

My telegram bot.

## Commands

|Command|Action|
|---|---|
|/help|Show this text|
|/say|Ask the bot to say something in a bot voice|
|/zhaw|Show current ZHAW stats|

## Getting Started

### Start

```
echo "BOT_SECRET_TOKEN" > src/secret.key
docker-compose up -d
```
